Fortgeschrittene API-Konzepte

Datenbank, Sicherheit und saubere API-Strukturen

1. Datenbankanbindung (SQLAlchemy)

Für reale Anwendungen reicht eine Liste nicht mehr aus. Mit SQLAlchemy kann eine echte Datenbank angebunden werden.

pip install flask-sqlalchemy
from flask_sqlalchemy import SQLAlchemy
db = SQLAlchemy()

class Book(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    title = db.Column(db.String(200))
    author = db.Column(db.String(100))

3. Datenvalidierung (Marshmallow)

pip install marshmallow flask-marshmallow

Validierung verhindert fehlerhafte oder unvollständige Daten.

  • Felder prüfen

  • Fehlermeldungen strukturieren

4. API-Dokumentation (Swagger)

pip install flasgger

Swagger erzeugt eine visuelle Dokumentation direkt im Browser.

  • Alle Endpunkte sichtbar
  • Testbar über UI

3. Was ist eine API?

APIs ermöglichen die Kommunikation zwischen Softwaresystemen. Eine REST-API verwendet HTTP-Methoden wie GET, POST, PUT oder DELETE, um Ressourcen zu lesen, anzulegen, zu ändern oder zu löschen.

GET

Daten abrufen

Beispiel: alle Bücher anzeigen.

POST

Daten anlegen

Beispiel: neues Buch hinzufügen.

PUT

Daten ändern

Beispiel: bestehendes Buch aktualisieren.

DELETE

Daten löschen

Beispiel: Buch aus der Liste entfernen.

5. Erste Flask-Anwendung

Der Einstieg beginnt mit einer minimalen Flask-Anwendung. Eine Route gibt eine einfache Antwort zurück.

Python · Flask 
from flask import Flask

app = Flask(__name__)

@app.route("/")
def hello_world():
    return "Hello, World!"

if __name__ == "__main__":
    app.run(debug=True)

6. Beispielhafte API-Endpunkte

Die Endpunkte zeigen, wie eine Ressource über verschiedene HTTP-Methoden verwaltet werden kann.

GET /api/books

Alle Bücher abrufen

POST /api/books

Neues Buch hinzufügen

PUT /api/books/<id>

Buch aktualisieren

DELETE /api/books/<id>

Buch löschen
